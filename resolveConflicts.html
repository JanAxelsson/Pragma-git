You are in this window because there are file conflicts. This could be because the file has been deleted in one place, or that there are differences in the same row. Both these cases needs a manual choice which change to keep.

This window has a stepwise approach :
Resolve the conflicts one section at a time. You may fold the sections, by clicking on the header.

Section "1) Conflicting files" :
Resolve the conflicts one section at a time. This is initiated by the "Solve conflicting files" button, which starts an external merge tool (see Requirements, below)

Section "2) Deleted files" :
Files that you are working on were deleted in the place you are merging from. Typically this is the intention to delete them (checked box), but in case you prefer to keep your files, uncheck the box.

Section "3) Accept changes" :
This is your last chance to change your mind.
Press "Undo Merge" in case you do not accept.
Press "Click to Accept" to finish. (This will also remove some backup-files created when working with the conflicts)
NOTE : After this you now can work on, or you wish to give a description and press the Store button in the main window

Requirements :
This window uses the external merge tool, which you define in the settings window.
