Git started

Intro

The aim of this app is to be a user-friendly revision control client. It fits people working on their own projects, or that find git overwhelming. The app uses git behind the scene, and is therefore fully compatible and can be used interchangeably together with other git softwares.

Drag and drop folder

Dragging a project's folder onto the app's window makes it available for version control. If the folder is not prepared (as a git repository), the user is asked to allow the program to initialize it for git.

Store version with description

The current states of the project's files are given a text description, and stored by clicking the "STORE" button.

Advanced

Repositories : You can have multiple active project folders (repositories). You switch by clicking the name (the first text in the title bar), to change to next project in the list.

Branches : You can have multiple branches of your work, that you can switch between. You switch by clicking the name (the second text in the title bar between parentheses), to change to next project in the list.
