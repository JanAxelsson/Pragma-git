About

Intro

The aim of this app is to be a user-friendly revision control client. It fits people working on their own projects, or that find git overwhelming. The app uses git behind the scene, and is therefore fully compatible and can be used interchangeably together with other git softwares.

Drag and drop folder

Dragging a project's folder onto the app's window makes it available for version control. If the folder is not prepared (as a git repository), the user is asked to allow the folder to be prepared.

Store version with description

The current states of the project's files are given a text description, and stored by clicking the "STORE" button. This is in contrast to the complicated (*) sequence of git commands normally used.

Acknowledgements

This program is written with NW.js, and the nw code is based on the nw-sample-apps examples.

Footnotes

(*) Store performs the following sequence of git commands :
- git add .
- git commit -all
- git pull (if remote repository is linked)
- git push(if remote repository is linked)
